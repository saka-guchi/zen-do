# ZenClip Documentation

ZenClip is a lightweight Windows desktop utility that instantly processes selected text with AI (via Gemini CLI) using a simple keyboard shortcut.

## Installation

1. Download the latest `ZenClip.zip` from [GitHub Releases](https://github.com/saka-guchi/zen-clip/releases).
2. Extract the archive to any folder and run `ZenClip.exe`.
3. The application will start and appear in your system tray.

## Basic Usage

1. Select (highlight) any text in your browser, code editor, or any application.
2. Press the default shortcut key: `<ctrl> + <shift> + c`.
3. Gemini CLI will process the text in the background, and the result will be automatically copied to your clipboard.

## Settings & Pro Activation

Right-click the ZenClip icon in the system tray and open "Settings".

- **General**: Customize your hotkey and toggle the auto-copy to clipboard feature.
- **Appearance**: Change the UI theme (Light/Dark) and language.
- **License (Pro Version)**: 
  1. Enter your `ZENCLIP-XXXX-XXXX` license key purchased via Polar into the License Key field.
  2. Click the "Activate" button to instantly unlock Pro features.
