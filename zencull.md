# ZenCull (Coming Soon)

ZenCull is a high-speed image culling tool for photographers, designed to help you instantly pick the "best shots" from massive amounts of RAW data using AI-assisted analysis.

## Project Status

ZenCull is currently **under active development**. We are working hard to bring the same "frictionless" experience of ZenClip to the photography workflow.

## Planned Features

- **AI-Assisted Selection**: Automatically group similar sequences and suggest the sharpest, best-composed shots.
- **Instant RAW Preview**: High-performance rendering engine for lag-free browsing of large RAW files.
- **Privacy First**: All image analysis and processing happen locally on your machine—no cloud uploads required.
